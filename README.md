# Django DevOps Lab

A production-ready Django project demonstrating how to integrate **Redis caching**, **Docker**, and **modern deployment practices** using **Render**.

This project is built as a hands-on learning lab for understanding real-world DevOps workflows with Django.

---

## 📌 Features

* Django 6 project setup
* Redis caching using `django-redis`
* Dockerized environment
* Production-ready setup with:

  * Gunicorn
  * WhiteNoise (static files)
* Deployment on Render

---

## 🛠️ Tech Stack

* **Backend:** Django
* **Cache:** Redis
* **Containerization:** Docker
* **Process Manager:** Gunicorn
* **Static Files:** WhiteNoise

---

## What I Learned

* How to integrate Redis with Django
* Dockerizing Django applications
* Preparing Django for production
* Deploying containerized apps on Render
